- Customizable 3D Hero Section
- Interactive Experience and Work Sections
- 3D Skills Section
- Consistent Animations
- Responsive Design

## Tech Stack

- Three.js
- React Three Fiber
- React Three Drei
- Framer Motion
- Email JS
- Tailwind CSS
